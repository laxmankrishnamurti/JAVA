# History of "Java"?

In Early 19's like - 1995, most of the processor has some different-different computational power for executing machine code. Because of the diversity some program runs on limited processor. So, it's very hard to write code that runs on most of the systems.

Here, comes "Java". The main feature of the programming language is that it dosen't compile the code into machine code directly but instead of doing this first of all it convert the code into "Byte Code". After that we can convert the "Byte Code" into the machine code that system required.

# Architecture of the Java Language.

- JDK (Java Development Kit) : It is the Superset of JRE & JVM
- JRE (Java RunTime Enviroment) : It provide the Enviroment to run Java Program
- JVM (Java Virtual Machine) : It convert the Byte code into Machine code.

# Miscellaneous

Hence, Java is an Object Oriented Language.

class : It's a Syntax to define a class <br>

void : Tells us what type of data we are returning from the class. In case, there is no any value is returning from the class so that's why it is void, means Empty. <br>
main : Method name (Without main, JVM does not know, from where i start executing the code. It is the main-gate of the class.) <br>
(String[] args) : Arguments <br>
